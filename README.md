# Rev'O Carrière - Website

This is the Next.js project for the Rev'O Carrière website, a platform for career transformation, digital products, and professional certifications.

## Stack

*   **Framework**: [Next.js](https://nextjs.org)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

*   [Node.js](https://nodejs.org/en/) (version 18.x or later)
*   [npm](https://www.npmjs.com/)

### Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory:
    ```bash
    cd revo-carriere
    ```
3.  Install the dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The pages will auto-update as you edit the files in the `src/app` directory.

## Project Structure

*   `src/app`: Contains all the pages of the website.
*   `src/components`: Contains all reusable React components.
*   `src/app/globals.css`: Contains the global styles and Tailwind CSS theme configuration.
*   `public`: Contains static assets like images and fonts (if not using `next/font`).
